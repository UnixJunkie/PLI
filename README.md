# Astex PLI #

### What is this repository for? ###

This repository contains the Astex PLI codebase for analysing and scoring protein-ligand interactions.

### How do I get set up? ###

* Create a directory <dir> to contain the PLI codebase. 
* Download and extract the PLI codebase into this directory.
* Type 'make' in <dir>, which should compile the code and create a binary <dir>/bin/pli.
* Set the environment variable PLI_DIR to <dir>.
* Try running one of the examples in the examples directory, e.g. by typing "<dir>/bin/pli -settings <dir>/examples/1a9u_contacts.pli"
